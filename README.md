# InventarioILS
Control de stock de inventario hecho con ~~.NET Framework 4.8~~ .NET 9.0 y WPF.

**Posibilidad de mejorar**
- Adaptar arquitectura a *MVVM* (parece ser la más adecuada en WPF, requeriría reescribir parte del código) y uso de *bindings* en la mayoria del código
- Copias de seguridad de la base de datos en línea
- Migración a una base de datos más robusta como *PostgreSQL*
- Migración a una tecnología .NET mas actual como *MAUI* (requeriría reescribir parte del código)
